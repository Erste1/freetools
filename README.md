# FreeTools

**Free, beautiful, and privacy-first online tools for everyone.**

A collection of 12+ useful tools that run entirely in your browser. No server, no tracking, no signup required.

![Preview](https://via.placeholder.com/800x400/0071E3/FFFFFF?text=FreeTools)

## Features

- **100% Client-Side** — All processing happens in your browser. No data leaves your device.
- **Beautiful Design** — Clean, modern UI inspired by Apple's design language.
- **Fully Responsive** — Works perfectly on desktop, tablet, and mobile.
- **Multi-Language** — Available in 10 languages (EN, RU, ES, DE, FR, PT, JA, ZH, KO, AR).
- **No Dependencies** — Pure HTML, CSS, and vanilla JavaScript. No frameworks needed.
- **Free Forever** — No premium tiers, no ads, no hidden costs.

## Tools Included

| Tool | Description |
|------|-------------|
| Image Compressor | Reduce image size without losing quality |
| Color Picker | Pick colors, get HEX, RGB, HSL values |
| Text Tools | Case convert, count words, remove duplicates |
| JSON Formatter | Format, validate, and beautify JSON |
| QR Generator | Generate QR codes for any text or URL |
| Password Generator | Create strong, secure passwords |
| Base64 Encoder | Encode and decode Base64 strings |
| Hash Generator | Generate MD5, SHA-1, SHA-256 hashes |
| Macro Calculator | Calculate calories and macros |
| Regex Tester | Test regular expressions in real time |
| Unit Converter | Convert between measurement units |
| Lorem Ipsum Generator | Generate placeholder text |

## How to Use

1. **GitHub Pages (Recommended):**
   - Fork this repository
   - Go to Settings → Pages
   - Select "main" branch
   - Your site will be live at `https://yourusername.github.io/FreeTools/`

2. **Local:**
   - Clone the repository
   - Open `index.html` in your browser
   - No server needed!
